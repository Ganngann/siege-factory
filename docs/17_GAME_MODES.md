# Game Modes — Siege Factory

## Modes prévus

### Campagne (PvE — prioritaire)

- Suite de niveaux/scénarios avec objectifs variés
- Survire à X vagues
- Difficulté progressive
- Tutoriel intégré dans les premiers niveaux

### Survival (PvE)

- Vagues infinies, difficulté croissante
- Score basé sur le nombre de vagues survécues
- Carte générée aléatoirement (seed)
- Pas d'objectif de fin — jusqu'à la mort

### Bac à sable (PvE)

- Ressources illimitées ou paramétrables
- Pas d'ennemis
- Créer l'usine parfaite sans pression
- Mode créatif / debug

### PvP (plus tard)

| Mode | Description |
|---|---|
| **Last Stand** | Chaque joueur défend sa base, les vagues attaquent tout le monde. Dernier vivant gagne. |
| **Annihilation** | Objectif : détruire le HQ adverse. Usine + armée. |
| **Economic Victory** | Premier à accumuler X ressources gagne. Pousse au commerce/vol. |
| **Coop vs Waves** | 2+ joueurs coopèrent contre des vagues plus dures. |

### Mode multijoueur

| Sous-mode | Pairs | Description |
|---|---|---|
| Local (écran partagé) | 2 | Chaque joueur a sa zone de carte |
| P2P online | 2-4 | Simulation déterministe |
| Coop | 2-4 | Tous contre les vagues |
| PvP ranked | 2 | Avec ELO et anti-triche |

## État actuel

- **M1-M5** : seul le bac à sable / survival est développé.
- **M6** : la campagne est ajoutée.
- **M7** : le multijoueur arrive.
- Les modes sont des plugins optionnels activés via GameState ou settings.
