# Game Modes — Siege Factory

## État actuel

Le jeu propose un seul mode aujourd'hui :

- **Survival TD** : 10 vagues, difficulté croissante, win à 10 vagues, game over si HQ détruit

Ce mode est un **scaffold** pour développer le socle technique. Il restera comme mode de jeu une fois la destination atteinte.

## Modes planifiés (destination)

### Campagne (PvE)
- Suite de niveaux/scénarios avec objectifs variés
- Tutoriel intégré
- Déblocages progressifs via tech tree

### Survival (PvE)
- Vagues infinies, difficulté croissante
- Score basé sur le nombre de vagues survécues
- Carte générée aléatoirement

### Bac à sable (PvE)
- Ressources illimitées ou paramétrables
- Pas d'ennemis (sauf si activé)
- Mode créatif / debug

### Multijoueur (PvP & Coop)
| Mode | Description |
|---|---|
| **Last Stand** | Chaque joueur défend sa base, vagues partagées |
| **Annihilation** | Détruire le HQ adverse |
| **Economic Victory** | Premier à accumuler X ressources |
| **Coop vs Waves** | 2+ joueurs contre des vagues plus dures |
